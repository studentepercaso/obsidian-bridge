# Obsidian Bridge 0.3.2 — Public preview

## English

This is the first public GitHub preview of Obsidian Bridge.

### Highlights

- Guided Windows installer with no administrator privileges.
- Visual per-vault folder picker in the Bridge Control Obsidian companion.
- Separate read and write permissions.
- Eight non-mutating read tools.
- Create and append through preview, explicit confirmation, and commit.
- Default-deny configuration with path, timeout, output, replay, and conflict controls.
- Public Codex marketplace entry and bilingual documentation.

### Install

Download **Obsidian-Bridge-Setup-0.3.2.zip**, extract it completely, and run **INSTALLA-OBSIDIAN-BRIDGE.cmd**. Do not use GitHub's automatically generated Source code archives as the installer.

Verify the downloaded assets with **SHA256-0.3.2.txt**. Start with a disposable vault or synthetic folder and keep an independent backup.

The installer and Bridge Control user interface are currently in Italian. English documentation is included.
