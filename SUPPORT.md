# Support

Obsidian Bridge is a community-maintained public preview.

- Read the [installation guide](docs/INSTALLATION.en.md) or the [Italian guide](docs/INSTALLATION.md).
- Search existing [issues](https://github.com/studentepercaso/obsidian-bridge/issues) before opening a new one.
- Use synthetic vault names and note contents in public reports.
- For vulnerabilities, follow [SECURITY.md](SECURITY.md) and use GitHub private vulnerability reporting.

Support is best-effort. This project is independent and is not affiliated with or endorsed by Obsidian or OpenAI.

## Italiano

Obsidian Bridge è un'anteprima pubblica mantenuta dalla community.

- Leggi la [guida di installazione](docs/INSTALLATION.md).
- Cerca tra le [segnalazioni esistenti](https://github.com/studentepercaso/obsidian-bridge/issues) prima di aprirne una nuova.
- Nelle segnalazioni pubbliche usa soltanto nomi di vault e contenuti sintetici.
- Per le vulnerabilità segui [SECURITY.md](SECURITY.md) e usa la segnalazione privata di GitHub.

Il supporto è fornito senza garanzia. Il progetto è indipendente e non è affiliato né approvato da Obsidian o OpenAI.
